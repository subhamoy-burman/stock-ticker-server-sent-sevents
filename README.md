# Stock Market Ticker with Server-Sent Events (SSE)

This project demonstrates a simple implementation of a stock market ticker using Server-Sent Events (SSE) with an ASP.NET Core Web API and a JavaScript client. The server continuously sends stock price updates to the client, which displays them in real-time.

## Features

- Real-time stock price updates using Server-Sent Events (SSE)
- ASP.NET Core Web API for streaming stock prices
- JavaScript client for displaying stock prices

## Prerequisites

- .NET 6 SDK
- Visual Studio or any other C# IDE
- Modern web browser (e.g., Chrome, Firefox, Edge)
